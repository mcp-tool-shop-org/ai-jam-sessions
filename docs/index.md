# AI Jam Sessions

An MCP server that teaches AI to play piano — and sing. 120 songs, 12 genres, real MIDI, piano + vocal tract synthesis.

## Key Features

- **120 songs across 12 genres** — Classical, Jazz, Blues, Pop, Rock, and more
- **Real MIDI sheet music** with teaching annotations
- **Piano engine** with 6 keyboard voices (grand, upright, electric, honkytonk, musicbox, bright)
- **Vocal tract synthesis** via Pink Trombone (soprano, alto, tenor, bass)
- **SVG piano roll** for visual feedback the AI can read back
- **Practice journal** that persists learning across sessions

## Install / Quick Start

```bash
npm install -g @mcptoolshop/ai-jam-sessions
```
